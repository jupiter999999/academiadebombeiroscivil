IMPORTANTE:
O arquivo package.json e a pasta app precisam aparecer na raiz do repositório.

Estrutura correta:
    app/
    components/
    lib/
    public/
    package.json
    package-lock.json
    next.config.ts
    proxy.ts
    vercel.json

NÃO deixe assim:
    meu-repositorio/academia_bombeiro_civil/app/

A menos que você configure a Root Directory para:
    academia_bombeiro_civil

CONFIGURAÇÃO NA VERCEL

1. Abra o projeto na Vercel.
2. Vá em Settings > General.
3. Procure Root Directory.
4. Se package.json e app estão na raiz do GitHub, deixe vazio ou use ./.
5. Se estão dentro de uma pasta chamada academia_bombeiro_civil,
   escolha essa pasta como Root Directory.
6. Framework Preset: Next.js.
7. Build Command: npm run build.
8. Output Directory: deixe vazio.
9. Install Command: npm install.
